
## Завдання 1 — ООП «Гроші та товар»

### Опис
Створити клас `Money` для роботи з грошима певної валюти.

### Вимоги до класу Money

**Інкапсульовані поля:**
- Ціла частина (долари/євро/гривні…)
- Копійки (0–99)

**Властивості з перевіркою значень (валідація)**

**Методи:**
- Метод виведення суми у форматі: `125.90 грн`
- Метод задання суми з перевіркою коректності

### Клас Product

З використанням `Money` створити клас `Product`:

**Поля:**
- Назва
- Ціна (Money)
- Тощо

**Методи:**
- `IncreasePrice` - збільшення ціни
- `DecreasePrice` - зменшення ціни
- Коректне оновлення копійок та переходу гривень/центів

### Демонстрація
Продемонструвати роботу класів у консолі (меню).

### Бонусне завдання
Реалізувати `Equals`/`GetHashCode()` у `Money` + коректне порівняння цін.

## Завдання 2 — ООП «Музичні інструменти»

### Опис
Створити базовий клас `MusicalInstrument` і похідні класи для різних музичних інструментів.

### Похідні класи
- `Violin` (Скрипка)
- `Trombone` (Тромбон)
- `Ukulele` (Укулеле)
- `Cello` (Віолончель)

### Вимоги до класів

Для кожного інструменту реалізувати:

**Методи:**
- `Show()` - Назва інструменту
- `Desc()` - Опис (характеристики)
- `History()` - Коротка історія походження

**Технічні вимоги:**
- Використати наслідування від `MusicalInstrument`
- Використати `override` для перевизначення методів
- Інкапсулювати дані: назва, опис, історія — тільки через конструктор або властивості

### Поліморфізм
Зробити поліморфний виклик для всіх інструментів у колекції:

```csharp
List<MusicalInstrument> orchestra = ...
```

## Завдання 3 — Структури «Десяткове число»

### Опис
Створити структуру `DecimalNumber` для роботи з десятковими числами та їх конвертацією в інші системи числення.

### Вимоги до структури DecimalNumber

**Поля/Властивості:**
- Поле або властивість для цілого числа

**Методи:**
- `ToBinary()` - конвертація в двійкову систему числення
- `ToOctal()` - конвертація в вісімкову систему числення
- `ToHex()` - конвертація в шістнадцяткову систему числення

Кожен метод повертає `string`, результат красиво відображається у консолі.
