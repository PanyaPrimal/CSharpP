# Calculator Solution

Консольний додаток калькулятора з підтримкою різних типів обчислень: базовий, науковий та програмістський.

## Структура проекту

Проект складається з трьох основних компонентів:

- **HW1-Calculator** - точка входу в програму
- **CalculatorLib** - бібліотека з реалізацією різних типів калькуляторів
- **CalculatorApp** - бібліотека з логікою консольного додатку

## Класи калькуляторів

### BasicCalculator

Базовий клас калькулятора з основними арифметичними операціями.

**Методи:**
- `Add(double a, double b)` - додавання
- `Subtract(double a, double b)` - віднімання
- `Multiply(double a, double b)` - множення
- `Divide(double a, double b)` - ділення (з перевіркою ділення на нуль)
- `SetLastResult(double result)` - встановлення останнього результату

**Властивості:**
- `LastResult` - отримання останнього результату обчислення

### ScientificCalculator

Науковий калькулятор, що наслідується від `BasicCalculator` і додає наукові функції.

**Додаткові методи:**
- `Pow(double baseValue, double exponent)` - піднесення до степеня
- `Sqrt(double value)` - квадратний корінь
- `Sin(double angleInRadians)` - синус
- `Cos(double angleInRadians)` - косинус
- `Tan(double angleInRadians)` - тангенс
- `Log(double value)` - натуральний логарифм
- `Log10(double value)` - десятковий логарифм

**Перевизначені методи:**
- `Divide(double a, double b)` - перевизначений метод з додатковими перевірками на дуже малі числа та логуванням. Використовує `base.Divide()` для виклику базової реалізації.

### ProgrammerCalculator

Програмістський калькулятор для роботи з різними системами числення та побітовими операціями. Наслідується від `BasicCalculator`.

**Методи конвертації:**
- `ToBinary(long value)` - перетворення в двійкову систему числення
- `ToHex(long value)` - перетворення в шістнадцяткову систему числення
- `ToOctal(long value)` - перетворення в вісімкову систему числення
- `FromBinary(string binaryString)` - перетворення з двійкової системи
- `FromHex(string hexString)` - перетворення з шістнадцяткової системи

**Побітові операції:**
- `And(long a, long b)` - побітове І
- `Or(long a, long b)` - побітове АБО
- `Xor(long a, long b)` - побітове виключне АБО
- `Not(long value)` - побітове НІ
- `LeftShift(long value, int shift)` - побітовий зсув вліво
- `RightShift(long value, int shift)` - побітовий зсув вправо

## Використання

1. Запустіть програму (HW1-Calculator)
2. Оберіть тип калькулятора:
   - `1` - Basic Calculator
   - `2` - Scientific Calculator
   - `3` - Programmer Calculator
3. Для виконання обчислення введіть `y`, для зміни типу калькулятора - `n`
4. Введіть числа та оберіть операцію

### Доступні операції

**Basic Calculator:**
- `+` - додавання
- `-` - віднімання
- `*` - множення
- `/` - ділення

**Scientific Calculator:**
Всі операції Basic Calculator плюс:
- `pow` - піднесення до степеня
- `sqrt` - квадратний корінь
- `sin` - синус (аргумент в радіанах)
- `cos` - косинус (аргумент в радіанах)
- `tan` - тангенс (аргумент в радіанах)
- `log` - натуральний логарифм
- `log10` - десятковий логарифм

**Programmer Calculator:**
Всі операції Basic Calculator плюс:
- `tobinary` - перетворення в двійкову систему
- `tohex` - перетворення в шістнадцяткову систему
- `and` - побітове І
- `or` - побітове АБО
- `xor` - побітове виключне АБО

## Технічні деталі

- **.NET версія:** 8.0
- **Мова:** C#
- **Архітектура:** Об'єктно-орієнтована з використанням наслідування
- **Обробка помилок:** Перевірка ділення на нуль, перевірка валідності аргументів для математичних функцій

## Особливості реалізації

- Використання наслідування для розширення функціоналу базового калькулятора
- Перевизначення методу `Divide` в `ScientificCalculator` з використанням `base` для виклику базової реалізації
- Збереження останнього результату в усіх типах калькуляторів
- Інтерактивний інтерфейс з можливістю зміни типу калькулятора під час роботи
